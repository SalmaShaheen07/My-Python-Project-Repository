# 🐍 Python Fundamentals Project — Data Technician Bootcamp (2026)
This project showcases the core Python programming skills I developed during the Data Technician Bootcamp (May–June 2026). Through a series of guided Google Colab exercises, logic‑building tasks, and hands‑on challenges, I learned how to write clean, structured Python code that interacts with users, performs calculations, and automates simple decision‑making.

## 🔧 Key Skills Demonstrated
### 1. Variables & Data Types
I used variables to store user inputs, numeric values, and program states across multiple exercises. Tasks included temperature conversion, area calculations, and arithmetic operations, as outlined in the workbook:

“Write a Python program to calculate the sum of two numbers entered by the user.”

### 2. print() and input() Functions
I built interactive programs that request information from users and display meaningful output. Examples include greeting messages, age prompts, and ATM menu navigation:

“Create a program that takes a user's name and age as input and prints a greeting message.”

### 3. Type Casting
I converted user inputs from strings to integers or floats to perform calculations such as compound interest, rectangle area, and number classification.

### 4. Conditional Logic (if, elif, else)
I applied conditional statements to control program flow, including:

Even/odd number checks

Positive/negative/zero classification

Leap year detection

FizzBuzz logic:

“If a number is divisible by 3, print ‘fizz.’ … If both divisible by 3 and 5, print ‘fizzbuzz.’”

These exercises strengthened my ability to build decision‑making into programs.

### 5. Loops (for and while)
I used loops to iterate through sequences, repeat menu options, and automate repetitive tasks. Examples include:

Printing the first 10 even numbers

Finding the largest number in a list

Running FizzBuzz from 1 to 100

Creating a loop‑driven ATM simulation with repeated menu prompts:

“Re‑use the ATM menu options across the program to ask the user if he/she would like to perform another operation.”

### 6. Building a Multi‑Step Program (ATM Simulation)
One of the most comprehensive tasks involved creating an ATM program that:

Displays a main menu

Shows balance

Handles deposits and withdrawals

Validates amounts

Uses functions for modularity

Repeats until the user exits

This project combined variables, loops, conditionals, input handling, and logic design into a single working application:

“Print a main menu… display balance, deposit money, withdraw money… exit from the ATM.”

## 📁 Project Structure
basics.py — simple input/output and arithmetic tasks

conditions.py — number checks, prime detection, leap year logic

loops.py — FizzBuzz, list processing, iterative tasks

atm_simulation.py — full ATM program with menu system and validation

exercises.ipynb — Google Colab notebook containing all completed tasks

These foundational skills prepare me for more advanced Python work in data analysis, automation, and scripting.
