# 🐍 Python Fundamentals Project — Data Technician Bootcamp (2026)
This project demonstrates the core Python programming skills I developed during the Data Technician Bootcamp (May–June 2026). Through a series of Google Colab exercises and logic‑building tasks, I learned how to write interactive Python programs using variables, user input, loops, and conditional statements. These exercises formed the foundation for building simple applications and performing calculations, as reflected in the workbook:

“Write a Python program to print ‘Hello, World!’… calculate the sum of two numbers… check if a number is even or odd…”
“Use loops and if statements.”

## 🔧 Key Skills Demonstrated
### 1. Variables & Type Casting
I used variables to store user inputs, numeric values, and program states. Many exercises required converting input strings into integers or floats to perform calculations, such as temperature conversion, area calculations, and digit sums.

### 2. Input & Output (input() and print())
I built interactive programs that request information from users and display meaningful results. Examples include greeting messages, age prompts, and multi‑step calculations.

“Create a program that takes a user's name and age as input and prints a greeting message.”

### 3. Conditional Logic (if, elif, else)
I used conditional statements to control program flow and build decision‑making into applications. Tasks included:

Even/odd checks

Prime number detection

Positive/negative/zero classification

Leap year validation

FizzBuzz logic

“If a number is divisible by 3, print ‘fizz’… If both divisible by 3 and 5, print ‘fizzbuzz.’”

### 4. Loops (for and while)
I used loops to repeat actions, iterate through lists, and automate logic. Examples include:

Printing the first 10 even numbers

Finding the largest number in a list

Summing positive numbers

Running FizzBuzz from 1 to 100

“Go through the integers from 1 to 100… Use loops and if statements.”

### 5. Building a Multi‑Step Program (ATM Simulation)
One of the most comprehensive tasks involved creating an ATM program that:

Displays a main menu

Shows balance

Handles deposits and withdrawals

Validates user input

Repeats until the user exits

This project combined variables, loops, conditionals, and user interaction into a single working application.

“Print a main menu… display balance, deposit money, withdraw money… exit from the ATM.”

## 📁 Project Structure
basics.py — simple input/output and arithmetic tasks

conditions.py — number checks, prime detection, leap year logic

loops.py — FizzBuzz, list processing, iterative tasks

atm_simulation.py — full ATM menu‑driven program

exercises.ipynb — Google Colab notebook containing all completed tasks
